# Cloned Repositories

## Repo 1: MAFALDA
- **URL**: https://github.com/ChadiHelwe/MAFALDA
- **Purpose**: Unified fallacy benchmark with refined taxonomy; evaluation framework for zero-shot fallacy detection
- **Location**: code/MAFALDA/
- **Key files**: datasets/gold_standard_dataset.jsonl, cli.py, src/, models/
- **Notes**: Contains evaluation scripts for running LLMs on fallacy detection. Requires GPU for some model evaluations.

## Repo 2: logical-fallacy (causalNLP)
- **URL**: https://github.com/causalNLP/logical-fallacy
- **Purpose**: Foundational fallacy detection dataset (LOGIC + LogicClimate) and model code
- **Location**: code/logical-fallacy/
- **Key files**: data/ (CSV datasets), codes_for_models/, codes_for_analysis/
- **Notes**: Contains both the dataset and training/evaluation code for fallacy classifiers.

## Repo 3: CoCoLoFa
- **URL**: https://github.com/Crowd-AI-Lab/cocolofa
- **Purpose**: LLM-assisted crowd-sourced fallacy dataset (7,706 comments, 8 types)
- **Location**: code/cocolofa/
- **Key files**: train.json, test.json, dev.json
- **Notes**: Dataset files directly in repo root.

## Repo 4: logical-fallacy-identification (USC-ISI)
- **URL**: https://github.com/usc-isi-i2/logical-fallacy-identification
- **Purpose**: Multi-method fallacy identification including CBR, data augmentation, propaganda detection, Prototex
- **Location**: code/logical-fallacy-identification/
- **Key files**: CBR/, fallacy_classification/, propaganda_detection/, prototex/, data/
- **Notes**: Contains implementations of multiple baseline methods.

## Repo 5: FALLACIES
- **URL**: https://github.com/Raising-hrx/FALLACIES
- **Purpose**: 232-type hierarchical fallacy dataset and evaluation framework for LLM self-verification
- **Location**: code/FALLACIES/
- **Key files**: step_fallacy.test.jsonl, fallacy_taxonomy.json, fallacies.json
- **Notes**: Contains the dataset, taxonomy, and detailed fallacy definitions.

## Repo 6: BIG-Bench-Mistake
- **URL**: https://github.com/WHGTyen/BIG-Bench-Mistake
- **Purpose**: CoT traces with annotated reasoning errors for testing mistake-finding ability
- **Location**: code/BIG-Bench-Mistake/
- **Key files**: *.jsonl (one per task: word_sorting, tracking_shuffled_objects, logical_deduction, multistep_arithmetic, dyck_languages)
- **Notes**: 2,186 traces generated by PaLM 2 Unicorn with human-annotated error locations.

## Repo 7: RuozhiBench
- **URL**: https://github.com/LibrAIResearch/ruozhibench
- **Purpose**: Bilingual benchmark for evaluating LLMs on logical fallacies and misleading premises
- **Location**: code/ruozhibench/
- **Key files**: data/ruozhibench_mc.jsonl, data/ruozhibench_gen.jsonl, src/
- **Notes**: 675 questions, bilingual (Chinese/English), with evaluation code.
